How Computer Works

Computers are machines that handles data. They can store, receive and process information. These electrical machines are programmable and are made to execute specific instructions that are programmed.

When the power is first turned on the computer, a signal is sent to the power supply and electricity and voltage is sent to the computer’s components. When the components are ready and no error signal is sent to the CPU, the CPU gets assigned a hexadecimal number that prepares it to receive instructions from the BIOS. After the computer turns on, the process of booting up begins. As the user provides input; the system software tells the CPU to start up programs and hardware devices so the computer is setup to collect more input. When the user makes a click on the mouse, a copy of the software program is copied into the RAM from the hard disk drive. Software is the set of instructions that are coded as tasks which the computer follows with the help of the hardware and the physical components of a computer, in order to take in input and produce output.

Motherboard

Important part of the computer. It holds many components of a computer. It’s a board made of non-conductive plastic with thin layers of copper or aluminum foil printed as narrow tracks. Many sockets are also made in order to hold in other components.

Parts of the motherboard:

  1. CPU socket that stores the CPU
  2. Fans mount next to CPU socket
  3. Power connector which distrivbutes power through the motherboard
  4. Memory slots for ram
  5. Video and graphics cards slotes such as Accelerated Graphics Port (AGP) and Peripheral Component Interconnect Express(PCIe)
  6. Northbridge and southbridge make up the chipset
    • Southbridge controls input and output functions
    • Northbridge forms an interface between main memory, CPU and other components
  7. Connectors to periperals(handled by Southbridge)
  8. ROM(read only memor also known as BIOS), with startup insturctions or firmware
    • BIOS is a non-violatile firware that give instructions during the startup process and boot the operating system
  9. Slots to connect hardware in the form of Peripheral Component Interconnect(PCI) slots
  10. Slots for hard drives to store files

CPU

The central processing unit (CPU) of a computer is a piece of hardware on the motherboard that handles the commands from other hardware and softwares. It performs the basic arithmetical, logical, and input/output operations of a computer system. The CPU is also called brain of the computer.

The main parts of a CPU include:

  1. The ALU that performs mathermatical and logical operations
  2. The CU that directs the processor operations

RAM

Ram or the Random Access Memory is a computer data storage device. This type of memory is different from ROM which is “read only memory”. ROM has coded instructions on how the computer runs thus it cannot be changed. It accesses data in a random order which makes it very fast to find to find a piece of information. The RAM is considered as volatile memory, meaning the information will be lost when power is off, meaning it is also “temporary memory”. The two main types of RAM include DRAM and SRAM. They are both volatile memories, The Static RAM is a lot faster and more expensive than Dynamic RAM.

Floppy disk

Computer hardware that reads data from the disk and writes to the disk. This outdated technology is now replaced with flash drives. This drive is a slot for floppy disks which a disk could be inserted and ejected. Floppy drives are installed with the connection in the computer and the slot facing outside the system unit. Connectors that lead to the motherboard and power supply are connected to the floppy disk drive.

Power supply

Power Supply Unit (PSU) a.k.a. power pack or power converter : This is a part of the computer which converts power from the outlet into usable power that is supplied to other parts in the system unit. Power supplies usually are mounted at the back of the system unit. It converts alternate-currents to direct-currents. It stabilizes heating automatically or manually. It has fans that release air out of the case. A PSU is important as the computer without power would not do any work. You would find an on and off switch on the power supply. There are also many wires that connect to the components of the computer that would need power such as fans, hard drives, etc. This component’s power is measured in watts to determine its power. Their models vary but it is never good to open the PSU. Sizes do matter in this case as they will differ between portables and desktops.

Hard disk drive

The hard disk drive is the dominant non volatile data storage hardware device in a computer, and permanently stores data on the computer. The hard disk drive use multiple rotating discs and rely on magnetic storage. Items such as the operating system, software titles and other files are stored here.

CD Drive

This device allows computers to read data off a compact disk. They cannot write data onto the disk. Microscopic bumps indicate 0 or 1 to the drive. A laser moves as the disk spins to detect indentations on the disk. A compact disc, a flat and round disk that stores data, can be accepted and ejected from this device.

Sound Card

An expansion card that provides output and input audio signals. They send signals to and from the computer through the control of certain programs. They have different ports such as microphone ports and digital out ports. Although many computers come with the sound card installed, they are not essential for the computer to function.

Voltage Regulator Module

The VRM - Voltage Regulator Module detects the amount of voltage in the microprocessor and helps it maintain a stable voltage. This is helpful when you change the microprocessor of a computer. In this case, the microprocessor can sense and accommodate to the new voltage level.
