How Computer Works

Computers are machines that handles data. They can store, receive and process information. These electrical machines are programmable and are made to execute specific instructions that are programmed.

When the power is first turned on the computer, a signal is sent to the power supply and electricity and voltage is sent to the computer’s components. When the components are ready and no error signal is sent to the CPU, the CPU gets assigned a hexadecimal number that prepares it to receive instructions from the BIOS. After the computer turns on, the process of booting up begins. As the user provides input; the system software tells the CPU to start up programs and hardware devices so the computer is setup to collect more input. When the user makes a click on the mouse, a copy of the software program is copied into the RAM from the hard disk drive. Software is the set of instructions that are coded as tasks which the computer follows with the help of the hardware and the physical components of a computer, in order to take in input and produce output.

Motherboard

Important part of the computer. It holds many components of a computer. It’s a board made of non-conductive plastic with thin layers of copper or aluminum foil printed as narrow tracks. Many sockets are also made in order to hold in other components.

    Parts of the motherboard
  1. CPU socket that stores the CPU
  2. Fans mount next to CPU socket
  3. Power connector which distrivbutes power through the motherboard
  4. Memory slots for ram
  5. Video and graphics cards slotes such as Accelerated Graphics Port (AGP) and Peripheral Component Interconnect Express(PCIe)
    6. Northbridge and southbridge make up the chipset
    • Southbridge controls input and output functions
    • Northbridge forms an interface between main memory, CPU and other components
  7. Connectors to periperals(handled by Southbridge)
    8. ROM(read only memor also known as BIOS), with startup insturctions or firmware
    • BIOS is a non-violatile firware that give instructions during the startup process and boot the operating system
  9. Slots to connect hardware in the form of Peripheral Component Interconnect(PCI) slots
  10. Slots for hard drives to store files

CPU

The central processing unit (CPU) of a computer is a piece of hardware on the motherboard that handles the commands from other hardware and softwares. It performs the basic arithmetical, logical, and input/output operations of a computer system. The CPU is also called brain of the computer.

    The main parts of a CPU include:
  1. The ALU that performs mathermatical and logical operations
  2. The CU that directs the processor operations
